Live Project: Restaurant Reservation

GitHub

This restaurant reservation is a front and back end application built to help restaurants keep track of reservations and tables.
Restaurant staff can use this application to create new tables, track capacity, availability and add/delete reservations based on date,
party size, and time. This is done by creating an SQL query for the PostgreSQL database.
Technologies used: React, Knex.js, PostgreSQL, JavaScript, HTML5, CSS.

Live Project: Decoder Ring

GitHub

Developed an application that encodes and decodes inputted messages.
Designed for easy use and can be used for writing encrypted messages or creating more secure passwords.
Uses the following ciphers: Caesar Shift, Polybius Square, and Substitution.
Technologies used: JavaScript, HTML5, CSS.
